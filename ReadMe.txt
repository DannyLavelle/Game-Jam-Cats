controls are simple enough 

the aim of the game is to cactch as many mice as possible 
to catch mice you need cats to assign to different tasks however cats need to stay motivated, they will lose motivation each time a task is complete 
when a cat loses motivation it will attempt to get some catnip, if you don't have any it will abandon you. 
these are the amount of motivation each task takes as a % of their base max motivation 
Catch mice 30%
Recruit Cat 100%
Gather Catnip 80%

spend mice in the upgrade shop on upgrades
